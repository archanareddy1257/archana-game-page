### Py Setup

#### create a Venv

cd backend
python -m venv webGames

#### Activate venv
.\webGames\Scripts\activate

#### install dependencies
pip install .\requirements.txt

### Web Setup

#### dependencies
npm i axios

## RUN

#### run in development 
npm run dev

#### build
npm run build
